# Calculator Web

A simple and responsive Calculator Web Application developed using HTML, CSS, and JavaScript. This project performs basic arithmetic operations with an interactive and user-friendly interface.

[Live Demo](https://khushalraut706.github.io/Caculator/)

## Features
*Basic Arithmetic Operations*
Addition
Subtraction
Multiplication
Division

## Tech Stack
HTML:Used for structuring the calculator layout and buttons.

CSS Used for:Styling the calculator,Responsive layout,Hover effects and animations,Modern UI design.

JavaScript:Implemented for:
Calculator logic
Arithmetic operations
Keyboard functionality
